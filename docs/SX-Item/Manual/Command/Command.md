# 指令

- `/si give [item] <player> <count> <key:value...>`
  - 给予玩家物品
- `si save [item] <type>`
  - 保存当前的物品到配置文件
- `si nbt <set/remove> <key> <value>`
  - 查看和修改物品NBT数据
- `si component <set/remove> <key> <json>`
  - 查看和修改物品组件数据
- `si script [file] [func] <args>`
  - 调用脚本函数
- `si reload`
  - 重新加载这个插件的配置
